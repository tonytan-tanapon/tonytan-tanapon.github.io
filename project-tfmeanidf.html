Term Weighting Algorithm (TFMeanIDF)

This project involved the development and publication of a novel term weighting algorithm (TFMeanIDF) optimized for imbalanced datasets. The algorithm significantly enhances machine learning model performance for natural language processing tasks.

Features

  • Optimized for imbalanced datasets
  • Improved term weighting calculation
  • Enhanced machine learning model performance
  • Validated using an Amazon dataset with over 10 million words across seven datasets

Technologies Used

  • Python
  • Natural Language Processing (NLP)
  • Machine Learning
  • Data Analysis

Publication

Published in An International Journal of Computing and Informatics, 2022.
