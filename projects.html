AI: Image classification using Tensorflow.js model (Tensorflow.js, Node.js, javascript, HTML, CSS)

The AI Image Classification feature allows users to upload an image and classify it using TensorFlow.js with the MobileNet model. When an image is uploaded, it is displayed in the interface, and the system processes it to generate predictions.

The model analyzes the image and returns the predicted class along with a confidence score. This feature demonstrates real-time AI inference within the browser without requiring a backend server, making it fast and efficient for classification tasks.

Features

  • Upload an image and classify it using TensorFlow.js (MobileNet model).
  • Display classification results on the UI.

Linear search animation (javascript, HTML, CSS)

The Linear Search feature visually demonstrates how the linear search algorithm works. It generates a set of random numbers displayed as bars in a chart. When a user enters a number to search, the system highlights each bar sequentially until it finds a match or reaches the end of the array.

The search process is animated with color changes, where the current bar being checked turns red, and if a match is found, it turns green. If the number is not found, a message is displayed. This feature helps users understand the step-by-step process of linear search interactively.

  • Generate an array of random numbers.
  • Perform a linear search on the array.
  • Animate the search process and display search results.
